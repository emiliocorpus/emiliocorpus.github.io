What is the difference between absolute, relative, and fixed positioning?

  • Absolute

    This type of positioning allows one to place any page element exactly where you want it. The positioning attributes you would want to use are top, left, bottom and right Theses values will be relative to the next parent with relative or absolute positiong. If there isn't a parent, then the object you are trying to postion will be relative to the html tag.


  • Relative

    Apparently, this type of positioning is the most confusing. From my understanding, this type will position itself in relativing to where it would be if you do not add any attributes. The attributes you will use are like absolute:
    top, bottom, left, and right.

  • Fixed

    With fixed positioning, elements are completely indepdent of everything else on teh web page. Regardless of any parents, a fixed position element will always be positioned base on the browswer window. This is the fixed scroller that people see on some pages.
